# 미니 던전 RPG

바람의나라 스타일 복고풍 2D 미니 던전 RPG 게임입니다.
Phaser 3 기반으로 제작되었으며, 게임 제작 흐름 학습을 목적으로 합니다.

## 주요 기능

- 타일맵 기반 마을/던전 탐험 (4방향 이동, 충돌 처리)
- 턴제 전투 시스템 (슬라임, 박쥐, 스켈레톤, 보스 어둠기사)
- NPC 대화 시스템 (퀘스트 부여, 선택지)
- 상점 및 인벤토리 (포션, 장비 구매/사용)

## 기술 스택

- **Phaser 3** (CDN)
- **Tiled** (타일맵 에디터)
- **Vanilla HTML/JS** (ES Modules)

## 실행 방법

빌드 도구 없이 로컬 웹 서버만 있으면 실행 가능합니다.

```bash
# 방법 1: Python
python3 -m http.server 8000

# 방법 2: Node.js (npx)
npx serve .

# 방법 3: VS Code Live Server 확장 사용
```

서버 실행 후 브라우저에서 `http://localhost:8000` 으로 접속하면 게임이 시작됩니다.

> ES Modules(`type="module"`)를 사용하므로 `file://` 프로토콜로는 실행되지 않습니다. 반드시 로컬 웹 서버를 통해 접속해주세요.

## 조작법

- **방향키**: 캐릭터 이동
- **Space / Enter**: 대화 진행, 메뉴 선택

## 게임 흐름

```
타이틀 → 마을 (NPC 대화 / 상점) → 던전 (몬스터 전투) → 보스전 → 엔딩
```
